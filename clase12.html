Proyecto final con integración Apps Script + IA

Objetivo

Demostrar cómo un proyecto puede combinar Google Workspace, IA y automatización.

Contenido práctico

  • Integrar todo lo aprendido en un proyecto real
  • Combinar Google Sheets, Apps Script y IA
  • Crear solución automatizada a problema personal/profesional
  • Documentar proceso con ayuda de IA

Opciones de proyecto

  • Sistema de inventario automatizado
  • Generador de facturas personalizadas
  • Control de gastos con reportes automáticos
  • Base de datos de clientes con seguimiento
  • Proyecto personalizado según necesidades

Requisitos del proyecto

  • Usar un script que haga algo útil (limpiar, calcular, generar, enviar)
  • Explicar qué hace el script con apoyo de IA
  • Mostrar cómo lo adaptarían para otros usos
  • Presentar solución completa al grupo
  • Documentar beneficios obtenidos

Producto final

Proyecto integrado que incluya:

  • Google Sheets como base de datos
  • Apps Script funcionando automáticamente
  • Documentación creada con ayuda de IA
  • Presentación de 5 minutos explicando la solución
